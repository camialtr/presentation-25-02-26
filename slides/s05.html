Atributos Essenciais

hx-get / post

Faz requisição AJAX para uma URL.

hx-get="/api/user"
hx-trigger

Define quando a requisição acontece.

hx-trigger="keyup changed delay:500ms"
hx-target

Define onde o conteúdo será inserido.

hx-target="#search-results"
hx-swap

Define como substituir o conteúdo.

hx-swap="outerHTML"
